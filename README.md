# Mercado Retro Catalog

A very small React + Vite + TailwindCSS site for showing products you sell on Mercado Libre.

## Run it

```bash
npm install
npm run dev
```

## Edit products

Products live in `src/data/products.json`.

Each product has:

- `id`
- `name`
- `price`
- `image`
- `description`
- `mercadoLibreUrl`
- `badge`

Use images from `public/products` or paste your own image URLs.

## Printable catalog / PDF

Open the site, click **Print Catalog**, then choose **Save as PDF** in your browser.
